# Console-Based Library Management System

## Project Description
A Java console-based application designed to manage library operations such as book tracking, member registration, borrowing, and data persistence using files.

## Objectives
- Apply Java fundamentals and OOP principles
- Implement file-based data storage
- Build a menu-driven console application
- Ensure clean code structure and readability

## Features
- Add, remove, and search books
- Register library members
- Borrow books with due date tracking
- Detect overdue books
- Generate library statistics
- Persistent storage using text files

## Technologies Used
- Java
- File I/O
- ArrayList
- Maven (project structure)
